# Kubogent - Enterprise AI Infrastructure Platform

Kubernetes + Agentic AI platform for managing LLMOps, MLOps, and AIOps on Amazon EKS.

## Modules

- **Cluster Management**: Provision and manage EKS clusters at scale
- **AIOps**: Full lifecycle from model building to deployment to monitoring
- **ML Engineering**: Workbench for model optimization, hyperparameter tuning, GPU benchmarking

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS v4
- React Router, Recharts, React Flow
- Lucide React icons

## Development

```bash
npm install
npm run dev
```
